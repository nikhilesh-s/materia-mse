# materia-mse
Materia MSE is a student-driven materials science platform featuring blogs, explainers, and interactive tools like the Materials Explorer. Built to make MSE accessible, modern, and engaging for students and enthusiasts through clean design, data-driven visuals, and practical engineering insights.

## Project Layout

- Next.js app lives in `MateriaMSE/`

## Local Development

```bash
cd MateriaMSE
npm install
npm run dev:5001
```

App runs at `http://localhost:5001`.

## Vercel

When importing this repo into Vercel:

- Framework preset: `Next.js`
- Root Directory: `MateriaMSE`
- Build Command: `npm run build`
- Install Command: `npm install`
